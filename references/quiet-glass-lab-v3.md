# Quiet Glass Lab v3

这是 `video-maker` 当前默认的视觉方向说明。目标不是“把页面做得像苹果截图”，而是借鉴 Liquid Glass 的层级逻辑，让 UI 主动为内容退后。

## 核心判断

Liquid Glass 最值得学的是：

- 把功能层从内容层里剥出来
- 让控件、导航、章节、切换以更轻的材质浮在上面
- 让真正的主信息直接待在内容层，而不是什么都塞进玻璃卡片

所以 `Quiet Glass Lab v3` 的基本原则不是“到处毛玻璃”，而是：

- 内容层优先
- 功能层玻璃化
- 临时解释层可进可退

## 三层结构

### Layer 0: 内容底板

承载：

- 主标题
- 主图 / 主结构图
- 这一页唯一想让观众记住的结论

规则：

- 尽量用实体深色底板或标准 material
- 不默认上玻璃效果
- 保证信息稳定、清楚、可读

### Layer 1: 玻璃功能层

承载：

- 章节
- 当前问题
- 视图切换
- 进度
- 主 action

规则：

- 透明度低
- 体积轻
- 始终比内容层更克制

### Layer 2: 临时解释层

承载：

- 一句话类比
- 术语拆解
- 对照提醒
- 关键 caveat

规则：

- 只在需要时出现
- 讲完就退
- 不长期霸占画面

## 视觉约束

- 不做假状态栏
- 不做整页拟态 iPhone 外壳
- 不把品牌色铺满整个面板
- 不让玻璃层承担大段正文
- 不让每一张卡片都发光

## 令牌建议

```css
:root {
  --bg-canvas: #0b0d0f;
  --bg-stage: #111416;
  --bg-content: rgba(18, 21, 24, 0.92);

  --glass-regular: rgba(255, 255, 255, 0.10);
  --glass-clear: rgba(255, 255, 255, 0.06);
  --glass-tinted: rgba(245, 248, 252, 0.16);
  --glass-stroke: rgba(255, 255, 255, 0.18);
  --glass-stroke-soft: rgba(255, 255, 255, 0.08);

  --glass-blur-regular: 28px;
  --glass-blur-large: 36px;
  --glass-sat: 140%;
  --glass-dim-on-media: rgba(0, 0, 0, 0.35);

  --accent: #d0f810;
  --accent-ink: #101010;
  --text-primary: rgba(255, 255, 255, 0.96);
  --text-secondary: rgba(255, 255, 255, 0.72);
  --text-tertiary: rgba(255, 255, 255, 0.52);
}
```

## 页面节奏

- `01-hook`：大内容板 + 轻章节条
- `02-04`：允许一块临时解释玻璃层
- `05-07`：内容图区继续放大，玻璃只保留必要辅助信息
- `08-choice`：玻璃进一步收敛成少量结论 pill / 选择器

## 复盘问题

如果页面看起来“很精致”，但你一眼说不出这一页最重要的信息是什么，那就说明 glass 用多了。
